# 本地图库

## 介绍

使用[cn_clip](https://github.com/OFA-Sys/Chinese-CLIP)实现的
- **以文搜图**：描述图片内容，从本地图库中寻找相似图片
- **以图搜图**：读取剪贴板内图片，从本地图库中寻找相似图片

CLI搜图效果预览：

## 使用方式

1. 配置config.json中的图库文件夹路径。
2. 运行 增加新图片.py 向数据库添加图片。
3. 运行 新图片特征.py 为图库图片计算特征。
 - 运行 剪贴板图到图1 .py 以图搜图 或 文到图1.py 以文搜图。
